M2Viz: Web Tool for Visualizing Genetic and Proteomic Variants

Overview

M2Viz is a web-based application that generates interactive lollipop plots to visualize genetic mutations and protein modifications (e.g., SNVs, PTMs, SAAVs) using sequence data from Ensembl and UniProt. It supports multiple data types and produces high-quality, publication-ready plots.

Responsibilities:

  • Led the backend development using Python and Django to handle data uploads, processing, and API integration.
  • Built RESTful APIs to fetch gene/protein sequences and annotations dynamically from Ensembl and UniProt.
  • Designed the data pipeline to normalize mutation/modification coordinates and prepare them for visualization.
  • Integrated a secure plotting system using R (trackViewer) and managed backend-to-frontend communication via JSON and Base64.
  • Ensured smooth frontend-backend interaction with React.js for a responsive and user-friendly interface.

Predictive Model for Type 2 Diabetes

Overview

Developed an advanced machine learning model to predict the likelihood of Type 2 Diabetes using metabolite profiles derived from blood samples. This project aimed to assist in early diagnosis and personalized treatment planning.

Responsibilities:

  • Data Preprocessing: Cleaned and normalized raw blood sample data for analysis.
  • Feature Selection: Identified key predictors to enhance model performance.
  • Model Training: Used RandomForest and Gradient Boosting for model training.
  • Performance Evaluation: Cross-validated metrics for accuracy, precision, recall, and F1-score.
