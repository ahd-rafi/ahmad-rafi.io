Highlighted Projects

Predictive Model for Type 2 Diabetes

Overview

Developed an advanced machine learning model to predict the likelihood of Type 2 Diabetes using metabolite profiles derived from blood samples. This project aimed to assist in early diagnosis and personalized treatment planning.

Responsibilities:

  • Data Preprocessing: Cleaned and normalized raw blood sample data for analysis.
  • Feature Selection: Identified key predictors to enhance model performance.
  • Model Training: Used RandomForest and Gradient Boosting for model training.
  • Performance Evaluation: Cross-validated metrics for accuracy, precision, recall, and F1-score.

Gene Resistance Classification

Overview

Applied machine learning techniques to classify gene resistance against various drugs, contributing to the understanding of genetic factors influencing drug efficacy.

Responsibilities:

  • Data Preparation: Processed and transformed genomic and pharmacological data for use in classification models.
  • Model Development: Developed and optimized classification models using algorithms such as Support Vector Machines and Neural Networks to classify gene-drug interactions.
  • Result Analysis: Analyzed model outputs to identify patterns and insights related to gene resistance, supporting drug development and resistance studies.

Head and Neck Cancer Data Analysis

Overview

Conducted a comprehensive analysis of data related to head and neck cancer, including the development of a specialized database and visualization tools.

Responsibilities:

  • Database Development: Designed and implemented the Head and Neck Drug Resistance Database using SQL and Django, facilitating detailed data storage and retrieval.
  • Visualization Tool: Created an interactive gene visualization tool using Django and Matplotlib to enable intuitive exploration of cancer-related data.
  • Efficiency Improvement: Streamlined data processing workflows, achieving a 50% reduction in processing time, and supported over 50 researchers and users with the optimized tools.
